# Os

Like Pythons os standard library. Limited at the moment.

## Usage

```python
import os


def pseudo_fn():
    # TODO
```

## Supported os functions

- `makedirs`
- `mkdir`
- `remove`
- `rmdir`
- `rename`

note: certain arguments for these functions may not be supported.

## Supported os.path functions

- `join`
- `exists`
- `isdir`
- `isfile`
- `dirname`
- `basename`
- `split`
- `abspath`
