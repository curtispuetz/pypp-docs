# Supported exception types

## Exception

- `Exception`
- `ValueError`
- `AssertionError`

## LookupError

- `LookupError`
- `IndexError`
- `KeyError`

## RuntimeError

- `RuntimeError`
- `NotImplementedError`

## OSError
- `OSError`
- `FileNotFoundError`
- `NotADirectoryError`
- `PermissionError`
- `FilExistsError`
